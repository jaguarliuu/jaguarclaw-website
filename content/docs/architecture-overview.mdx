JaguarClaw is organized around a deterministic runtime loop:

- context preparation
- model invocation
- tool execution
- event emission
- completion/continuation decision

This model enables auditability and consistent behavior under operational constraints.

```ts title="runtime-loop.ts"
type LoopState = 'continue' | 'stop'

export function runStep(step: number, maxSteps: number): LoopState {
  if (step >= maxSteps) return 'stop'
  return 'continue'
}
```
