## Install

Follow the runtime setup guide, then launch JaguarClaw with your preferred provider configuration.

## First Run

1. Open workspace.
2. Configure model provider.
3. Execute a simple prompt with tool calls disabled first.

```bash title="quick-start.sh"
#!/usr/bin/env bash
set -euo pipefail

cd jaguarclaw-website
npm install
npm run dev
```
