# JaguarClaw 官网重构方案

**目标**：打造一个充满活力、有科技感、中英双语的企业级 AI 桌面助手官网

---

## 一、现状分析

### 1.1 当前问题

| 问题 | 描述 |
|------|------|
| 🎨 **风格过于冷淡** | 深色系 + cyan 配色虽然专业，但缺乏活力，像企业内网系统 |
| 📝 **纯英文内容** | 没有中文支持，对国内用户不友好 |
| 🏠 **首页缺乏冲击力** | Hero 区域文案过长，缺少视觉焦点和动态元素 |
| 🔧 **特性展示平淡** | 3 个特性卡片过于抽象，与 JaguarClaw 的实际功能脱节 |
| 🖼️ **缺少视觉元素** | 无动画、无插图、无截图，纯文字堆砌 |
| 📱 **交互单一** | 无任何微交互或动效，体验枯燥 |

### 1.2 保留优势

- ✅ 技术栈成熟：Next.js + Contentlayer + TypeScript
- ✅ 响应式布局已就绪
- ✅ 代码高亮和 MDX 支持完善
- ✅ 文档和博客架构清晰

## 二、设计方向

### 2.1 核心定位

**"AI 时代的工作伙伴"** —— 不是冰冷的工具，而是充满活力、智能、可靠的助手

### 2.2 美学方向

| 维度 | 当前 | 目标 |
|------|------|------|
| **情绪** | 冷静、专业 | 活力、智能、可靠 |
| **色彩** | 单一 cyan | 渐变 + 多彩点缀 |
| **动效** | 无 | 丰富的入场动画和微交互 |
| **排版** | 居中对称 | 不对称、动态布局 |
| **语言** | 英文 | 中英双语 |

### 2.3 品牌色彩体系

```css
/* 主色 - 从冰冷 cyan 到活力渐变 */
--primary-start: #00d4aa;     /* 翠绿 - 活力 */
--primary-end: #7c3aed;       /* 紫色 - 智能 */

/* 强调色 */
--accent-orange: #ff6b35;     /* 行动/CTA */
--accent-blue: #3b82f6;       /* 链接/交互 */
--accent-gold: #fbbf24;       /* 高亮/成就 */

/* 背景 - 保持深色但更有层次 */
--bg-base: #0a0a0f;           /* 深空黑 */
--bg-elevated: #14141f;       /* 卡片背景 */
--bg-surface: #1a1a2e;        /* 面板背景 */

/* 文字 */
--text-primary: #f8fafc;
--text-secondary: #94a3b8;
--text-muted: #64748b;
```

### 2.4 字体选择

```css
/* 标题 - 有力量感但不失现代 */
--font-display: 'Space Grotesk', 'Noto Sans SC', sans-serif;

/* 正文 - 清晰易读 */
--font-body: 'Inter', 'Noto Sans SC', sans-serif;

/* 代码 */
--font-mono: 'JetBrains Mono', monospace;
```

## 六、技术决策

### 6.1 新增依赖

```json
{
  "dependencies": {
    "next-intl": "^3.x",           // 国际化
    "framer-motion": "^11.x",      // 动画库
    "lucide-react": "^0.x"         // 图标库
  }
}
```

### 6.2 动画方案

- **简单动画**: CSS + keyframes
- **复杂交互**: Framer Motion
- **背景效果**: Canvas / CSS gradient + blur

### 6.3 图标方案

使用 Lucide React（现代、一致、轻量）
